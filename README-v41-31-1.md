# Beyond OS v41-31.1 — 멘토링 배정 UX 보정 패치

기준선: v41-31

## 반영 내용

1. 학생 복수 선택 UX 개선
   - 기존 native multiple select 방식은 Ctrl/Shift 또는 모바일 터치 방식이 헷갈릴 수 있어 클릭형 학생 카드 선택으로 변경했습니다.
   - 학생 카드를 클릭하면 선택/해제됩니다.
   - 선택 요일 중 이미 배정된 학생은 비활성 카드로 표시됩니다.

2. 개인 시간표 충돌 경고 가시성 개선
   - 배정 저장 전 개인 시간표와 맞지 않는 배정이 있으면 화면 하단 안내문만 띄우지 않고 중앙 상세 팝업으로 표시합니다.
   - 충돌 건수, 학생 수, 차시 수를 요약 표시합니다.
   - 학생별 기준일, 멘토링 차시, 개인 시간표, 시간표 메모, 충돌 사유를 상세 표시합니다.
   - 관리자가 내용을 확인한 뒤 `경고 확인 후 그래도 배정`을 눌러야 저장됩니다.

3. 서버 오류 데이터 보존
   - API 오류 응답에 포함된 conflicts payload를 프론트에서 읽을 수 있도록 보강했습니다.

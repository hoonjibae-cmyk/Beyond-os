# Beyond OS v41-32 — 출결·관리 이력 통합 및 멘토링 카드 연결

## 기준선

- 기준 버전: v41-31.4
- 목적: 왼쪽 메뉴 구조를 정리하고, 멘토링 시간표에서 학생 관리 이력으로 자연스럽게 이어지도록 개선

## 반영 내용

1. 왼쪽 메뉴 통합
   - 기존 `출결 현황` 메뉴와 `학생 관리 이력` 메뉴를 `출결·관리 이력` 메뉴 하나로 통합했습니다.
   - 통합 메뉴 안에서 `출결 현황 · 멘토 코멘트`와 `학생 관리 이력`을 내부 전환 버튼으로 선택할 수 있습니다.
   - 기존 `출결 현황` 직접 접근이 남아 있는 계정/상태가 있어도 `출결·관리 이력`으로 자동 이동하도록 보정했습니다.

2. 멘토링 시간표 학생 카드 연결
   - 멘토링 시간표의 차시 카드 안에 배정된 학생 이름 영역을 클릭하면 해당 학생의 `출결·관리 이력` 메뉴로 이동합니다.
   - 이동 시 기본 화면은 `출결 현황 · 멘토 코멘트`입니다.
   - 오늘 날짜 기준으로 해당 학생의 출결 현황을 자동 조회합니다.

3. 오늘 학습멘토 코멘트 입력 흐름 개선
   - 멘토링 시간표에서 학생을 클릭해 이동하면 `오늘 학습멘토 코멘트` 입력 영역으로 스크롤 및 포커스가 이동합니다.
   - 멘토가 시간표에서 학생 확인 → 해당 학생 코멘트 입력으로 바로 이어질 수 있도록 했습니다.

4. 권한 호환 보정
   - 기존 계정의 권한값에 `출결 현황` 권한만 남아 있어도 통합된 `출결·관리 이력` 메뉴 접근이 가능하도록 보정했습니다.
   - 신규 권한 설정 화면에는 통합 메뉴만 표시됩니다.

# Beyond OS v41-36.2

## 키오스크 쉬는 시간 HOLD 운영 안정화

기준선: v41-36.1

### 반영 내용

1. 같은 학생의 쉬는 시간 외출·복귀 신호를 한 묶음으로 표시합니다.
   - 예: `09:53 외출 → 09:59 복귀`
   - 한 학생의 묶음을 실제 출결로 일괄 반영하거나 쉬는 시간 이동으로 한 번에 처리할 수 있습니다.

2. 같은 학생의 같은 신호가 짧은 시간에 반복되면 첫 신호만 HOLD합니다.
   - 설정 위치: `설정 → 키오스크 브릿지 → HOLD 중복 신호 판단 시간(초)`
   - 기본값 30초, 설정 범위 5~120초

3. HOLD 목록을 날짜와 쉬는 시간 구간별로 묶습니다.
   - 구간 전체 선택 및 `이 구간 전체 쉬는 시간 처리` 지원

4. 외출만 있거나 복귀만 있는 미완결 신호를 주황색으로 우선 표시합니다.

5. HOLD 처리 이력을 표시하고 처리 후 10분 동안 되돌릴 수 있습니다.
   - 실제 출결 반영 되돌리기 시 Beyond OS 출결 상태는 복원됩니다.
   - 이미 발송된 학부모 알림은 취소되지 않습니다.

# Beyond OS v40-121

기준 파일: beyond-os-v40-120-sms-web-sender-plain-text-bridge.zip

## 변경 사항

- 키오스크 SMS/Web발신 파서 문구 확장
- 다음 문구를 추가 지원합니다.
  - `OOO 학생이 학원에 도착했어요.` → 입실
  - `OOO 학생이 잠시 외출했어요.` → 외출
  - `OOO 학생이 다시 돌아왔어요.` → 복귀/재입장
  - `OOO 학생이 하원했어요.` → 퇴실
- 퇴실 문자 재원시간이 `09시00분 ~ 21시50분` 형태로 와도 원문 로그와 duration에 보존합니다.
- 기존 `입장했어요`, `재입장했어요`, `퇴장했어요`, `입장했습니다`, `퇴장했습니다` 형식도 계속 지원합니다.

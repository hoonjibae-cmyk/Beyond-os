# Beyond OS v41-29 — Student History Polish & GPT Counseling Modes

## 기준선
- v41-28 `beyond-os-v41-28-student-management-history-gpt-summary.zip`

## 반영 내용

### 1. 학생 관리 이력 UI 정리
- 학생 관리 이력 화면의 날짜별 압축 관리표 영역을 모바일/태블릿에서 더 안정적으로 스크롤되도록 개선했습니다.
- 테이블 컨테이너에 모바일 터치 스크롤, 가로 스크롤, 최소 너비 보정을 적용했습니다.
- 표 하단에 “좌우로 스크롤 가능” 안내를 표시합니다.

### 2. GPT 상담 요약 2종 분리
- GPT 상담 요약 타입을 다음 2개로 정리했습니다.
  - 학생 피드백용: 직설적인 관리 포인트 중심
  - 학부모 상담용: 완화된 표현과 개선 방향 중심
- 운영자가 상담 내용을 수정/저장하는 흐름은 제거하고, 구두 상담 활용을 위한 생성/다시 생성/복사 중심으로 변경했습니다.

### 3. GPT 사용 안전장치
- `OPENAI_API_KEY` 미설정 시 화면에서 안내합니다.
- 조회 데이터가 부족하면 요약 생성을 차단합니다.
- 조회기간이 31일을 초과하면 생성 전 확인창을 띄웁니다.
- 생성 실패 사유를 화면에 표시합니다.

### 4. 관련 화면 바로가기
- 학생 관리 이력 화면에서 다음 화면으로 바로 이동할 수 있는 버튼을 추가했습니다.
  - 데일리 리포트
  - 위클리 리포트
  - 관리주의 이력
  - 알림/템플릿 설정

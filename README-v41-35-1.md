# Beyond OS v41-35.1

## 차시별 순찰 미입력 알림 보정본

기준선: v41-35

### 반영 내용

1. 좌석배치도 `미입력` 알림을 차시 단위로 판정
   - 기존: 오늘 순찰 기록이 1건이라도 있으면 이후 차시에도 `미입력` 알림이 사라질 수 있음
   - 변경: 현재 시간이 포함된 차시에 순찰 기록이 없으면 `미입력` 알림을 다시 표시
   - 예: 1차시 기록이 있어도 2차시가 시작된 뒤 2차시 기록이 없으면 다시 `미입력` 표시

2. 순찰 퀵체크의 미입력 학생 선택을 현재 차시 기준으로 변경
   - `미입력 n명 선택` 대상은 현재 차시에 기록이 없는 학생
   - 순찰자가 차시별 입력 누락 학생을 한 번에 선택 가능

3. 실시간 학습 분포를 현재 차시 기록 기준으로 집계
   - 현재 차시에 기록이 없는 학생은 분포 집계에서 미입력으로 처리

### 반영하지 않은 내용

- 좌석배치도의 학습상태 문구를 현재 차시 기준으로 강제 변경하지 않음
  - 좌석 본문에는 기존처럼 오늘 입력된 최신 순찰 기록을 표시
- 좌석 좌상단 배지는 기존처럼 `미입력`으로만 표시
  - `2차시 미입력`처럼 차시명을 배지에 붙이지 않음

### 추가 작업

- 신규 SQL 없음
- 환경변수 변경 없음
- MacroDroid 변경 없음

# Beyond OS v41-34.1 — 멘토링 기본 시간표 동기화 핫픽스

멘토링 차시 정보가 초기 하드코딩값이 아니라 설정 탭의 기본 시간표 1~8차시를 기준으로 동작하도록 보정한 핫픽스입니다.

## 변경 사항

- 멘토링 시간표의 차시 추가/수정 드롭다운이 `기본 시간표 설정`의 1~8차시를 불러옵니다.
- `멘토링 기본 설정`의 기본 세팅 버튼은 이제 기존 월/수/금 차시를 현재 기본 시간표 기준으로 갱신합니다.
- 기본 시간표 저장 시 멘토링 기본 차시도 자동 동기화됩니다.
- 날짜별 일정의 자동 반영 차시는 요일 템플릿과 동기화되고, 날짜별 직접 수정 차시는 보존됩니다.

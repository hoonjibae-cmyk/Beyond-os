# Beyond OS v41-36 — 쉬는 시간 키오스크 HOLD

기준선: v41-35.2

## 핵심 변경
- 설정 탭의 기본 시간표 차시 사이 공백을 쉬는 시간 HOLD 구간으로 자동 계산합니다.
- HOLD 범위는 각 차시 종료 시각부터 다음 차시 시작 1분 뒤까지입니다.
- 이 구간에 들어온 키오스크 입실·외출·복귀·퇴실 신호는 즉시 출결에 적용하지 않습니다.
- 대시보드 상단의 `쉬는 시간 키오스크 HOLD` 목록에서 관리자가 확인합니다.
- 실제 출결로 판단되는 항목만 `실제 출결 반영`으로 전환할 수 있습니다.
- 일반 쉬는 시간 이동은 개별 삭제 또는 선택 일괄 삭제할 수 있습니다.
- HOLD 상태에서는 좌석 상태, 순공시간, 출결 이벤트, 학부모 알림이 변경되지 않습니다.

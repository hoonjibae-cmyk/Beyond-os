# Beyond OS v41-33.3

## 목적
v41-33.2 날짜별 멘토링 일정 화면에서 좁게 보이거나 줄바꿈이 어색했던 컨트롤 폭과 정렬을 다듬은 UI 핫픽스입니다.

## 수정 내용

### 1. 날짜별 일정 상단 컨트롤 폭 보정
- `기준 날짜` 라벨이 두 줄로 깨지지 않도록 nowrap 및 최소 폭을 적용했습니다.
- 이전날/오늘/다음날/수정 시작/되돌리기/차시 추가 버튼의 폭과 padding을 조정했습니다.
- 날짜별 상태 배지는 화면을 과도하게 차지하지 않도록 최대 폭과 말줄임 처리를 적용했습니다.

### 2. 차시 카드 상단 배지 정렬 보정
- 차시 카드 우측의 `배정 부족 · 권장 3~4명`과 `요일 기본값/날짜별 수정` 배지가 서로 밀리거나 겹쳐 보이지 않도록 카드 헤더를 2열 grid 구조로 보정했습니다.
- 작은 카드에서도 배지가 오른쪽에 안정적으로 정렬되도록 폰트와 padding을 줄였습니다.

### 3. 날짜별 차시 수정/추가 에디터 폭 보정
- `적용 날짜`, `추가/변경할 차시`, `선택 시간`, `권장 최소/최대 인원` 필드 폭을 재조정했습니다.
- 차시 선택 드롭다운이 너무 좁게 보여 화살표만 남는 현상을 줄이도록 최소 폭을 늘렸습니다.
- 권장 인원 입력칸은 숫자를 가운데 정렬해 의미가 더 분명하게 보이도록 했습니다.

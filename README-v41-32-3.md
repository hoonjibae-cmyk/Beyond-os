# Beyond OS v41-32.3 — 멘토링 이동 기본 1주 조회 및 순공 기준 요약 가이드

## 기준선

- 기준 버전: v41-32.2
- 패치 성격: 멘토링 시간표 → 출결·관리 이력 이동 시 조회기간 보정, GPT 상담 요약 순공 기준 보강

## 반영 내용

1. 멘토링 시간표에서 학생 카드 클릭 시 기본 조회기간 1주 적용
   - 기존에는 멘토링 시간표에서 학생을 클릭하면 `오늘` 기준으로 출결·관리 이력이 열렸습니다.
   - 이제는 해당 학생이 자동 선택되면서 조회기간이 `최근 1주(오늘 포함 7일)`로 설정됩니다.
   - 오늘 학습멘토 코멘트 입력 포커스는 기존처럼 유지되므로, 오늘 코멘트 작성 흐름은 그대로 이어집니다.

2. GPT 상담 요약에 방학 순공 기준 반영
   - 상담 요약 생성용 데이터에 `studyVolumeGuide`를 추가했습니다.
   - 기준은 `1주 환산 순공시간`입니다.
   - 40시간 초과: `학습량 충분`
   - 30시간 이상 40시간 이하: `보통`
   - 30시간 미만: `개선 필요`
   - 조회기간이 7일이 아닌 경우에도 1주 환산값을 기준으로 판정하도록 보강했습니다.

3. GPT 프롬프트 보강
   - 학생 피드백용/학부모 상담용 모두 위 순공 기준을 상담 요약에 반영하도록 안내했습니다.
   - 조회기간 핵심 요약 안에 순공시간 판정과 근거 시간이 1회 포함되도록 가이드했습니다.

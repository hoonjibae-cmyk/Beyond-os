# Beyond OS v41-11 Dashboard Seat Map Realtime UI

기준선: `beyond-os-v41-10-attendance-notification-stability-console.zip`

## 핵심 변경

1. 기존 좌석 배치도 구조는 유지하되 메인 대시보드의 실시간 현황 가독성을 개선했습니다.
2. 좌석 상태별 필터를 추가했습니다.
   - 전체
   - 입실
   - 외출
   - 미입실
   - 퇴실
   - 결석
   - 관리필요
3. 학생명·좌석번호·상태 검색을 추가했습니다.
4. 좌석 카드에 상태, 학생명, 현재 과목/학습상태를 더 직접적으로 표시합니다.
5. 실시간 학습 분포 카드를 추가했습니다.
   - 현재 입실 학생 기준 학습상태 분포
   - 현재 입실 학생 기준 과목 분포
6. 관리 포커스 카드를 추가했습니다.
   - 외출 중
   - 결석
   - 관리필요
   - 학습상태 미입력
   - 미입실 학생
7. 리스트 보기에서도 입실/퇴실/외출 시각, 기록방식, 관리 이슈를 함께 확인할 수 있게 했습니다.

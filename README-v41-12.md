# Beyond OS v41-12 — Study Quick Check Patrol Mode

기준선: `beyond-os-v41-11-dashboard-seat-realtime-ui.zip`

## 업데이트 목적

v41-11에서 개선한 좌석 배치도와 실시간 현황 UI를 바탕으로, 관리자가 순찰 중 학습상태를 더 빠르게 입력할 수 있도록 개선했습니다.

## 주요 반영 사항

### 1. 좌석 카드 기반 순찰 퀵체크

- 메인 대시보드에 `순찰 퀵체크` 콘솔을 추가했습니다.
- `순찰 모드`를 켜면 좌석 클릭이 상세 패널 열기가 아니라 다중 선택으로 동작합니다.
- 입실 상태인 좌석만 퀵체크 일괄 저장 대상이 됩니다.

### 2. 다중 좌석 일괄 입력

- 여러 좌석을 선택한 뒤 한 번에 같은 과목/학습상태를 저장할 수 있습니다.
- 예: 3, 4, 5, 6번 좌석 선택 → `수학 / 문제풀이` → 일괄 저장
- 기존 `/api/check` 저장 흐름을 사용하므로 별도 신규 테이블은 필요 없습니다.

### 3. 과목/학습상태 빠른 선택 버튼

- 과목: 수학 / 영어 / 국어 / 사탐 / 과탐 / 기타
- 학습상태: 인강 / 문제풀이 / 암기 / 독서 / 수면 / 비학습
- 기존 드롭다운은 유지하면서, 태블릿 순찰에 맞는 큰 버튼형 선택 UI를 추가했습니다.

### 4. 최근 입력값 재사용

- 최근 사용한 과목/학습상태/내용 조합을 로컬 브라우저에 저장합니다.
- 다음 순찰 시 최근 사용 버튼으로 빠르게 재적용할 수 있습니다.
- 이 값은 브라우저 localStorage에 저장되며 Supabase 신규 테이블은 사용하지 않습니다.

### 5. 학습상태 미입력 학생 강조

- 입실 중인데 현재 과목/학습상태가 없는 좌석은 `미입력` 배지와 강조 테두리로 표시됩니다.
- 순찰 모드에서 `미입력 n명 선택` 버튼으로 한 번에 선택할 수 있습니다.

### 6. 기존 우측 패널 학습 체크 입력 개선

- 기존 학습 체크 입력 영역은 유지했습니다.
- 드롭다운 외에 과목/학습상태 버튼형 선택 UI를 추가했습니다.

# Beyond OS v41-02

## 기준

- 기준 파일: beyond-os-v41-01-kiosk-bridge-safety-mode.zip
- 신규 버전: v41-02
- v40 계열: v40-123에서 종료
- v41/v42 실험 브랜치: 미포함
- 신규 SQL: 없음

## 핵심 변경

### 1. 동일 기기 작업의 원격 변경 알림 억제

기존에는 같은 브라우저에서 출결/학생정보/시간표 등을 저장한 뒤 실시간 동기화가 다시 실행되면서 `다른 기기 변경 반영` 팝업이 과도하게 표시될 수 있었습니다.

v41-02에서는 브라우저별 clientId와 최근 로컬 변경 억제 시간을 활용해, 현재 기기에서 발생한 저장 작업 직후의 동기화는 원격 변경 알림에서 제외합니다.

### 2. 진짜 외부 변경만 알림 표시

최근 로컬 저장 작업이 없는 상태에서 대시보드 좌석/출결 signature가 바뀌면 기존처럼 `다른 기기 변경 반영` 알림을 표시합니다.

### 3. 원격 변경 알림 자동 닫힘

닫기 버튼을 누르지 않아도 약 5.2초 후 자동으로 사라집니다. 동일 signature 알림은 짧은 시간 안에 반복 표시되지 않습니다.

### 4. 출결 상태 버튼 체감 지연 개선

출결 상태 변경 후 중복 클릭 방지 잠금 시간을 기존 1.4초에서 약 0.45초로 축소했습니다. 저장 중에는 안전하게 잠금이 유지되지만, 저장 완료 이후 불필요한 랙은 줄였습니다.

### 5. 버전 표시 갱신

상단 버전과 `/api/health` 버전이 v41-02로 표시됩니다.

## 테스트

1. 상단 버전이 v41-02인지 확인
2. 현재 PC에서 출결 상태를 변경했을 때 `다른 기기 변경 반영` 팝업이 뜨지 않는지 확인
3. 학생 정보/시간표 저장 후에도 같은 팝업이 뜨지 않는지 확인
4. 다른 PC 또는 다른 브라우저에서 출결을 변경했을 때 원격 변경 알림이 뜨는지 확인
5. 원격 변경 알림이 약 5초 뒤 자동으로 사라지는지 확인
6. 출결 버튼 저장 완료 후 다음 클릭 가능 체감 시간이 짧아졌는지 확인

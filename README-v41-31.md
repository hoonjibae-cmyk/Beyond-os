# Beyond OS v41-31 — 멘토링 시간표 반복·복수배정 보정

기준선: v41-30

## 반영 내용

1. 멘토링 차시 범위 확대
   - 기존 4~8차시 중심 → 1~8차시 전체 기본 세팅
   - 월/수/금 각각 1~8차시 기본 차시 생성

2. 반복 설정 추가
   - 선택 요일만 반복
   - 월·수·금 반복
   - 직접 선택 반복
   - 배정은 주간 반복 기준으로 유지되며, 필요 시 기존 배정을 삭제하고 재배정

3. 학생 복수 선택 배정
   - 학생 선택 UI를 복수 선택으로 변경
   - 한 번에 여러 학생을 같은 차시/멘토에 배정 가능

4. 배정 저장 전 개인 시간표 충돌 경고
   - 배정 저장 전 학생 개인 시간표와 멘토링 차시를 비교
   - 멘토링 차시가 예정 학습실 체류 시간 밖이거나 외출 시간과 겹치면 상세 경고 팝업 표시
   - 관리자가 확인 후 `그래도 배정`을 선택해야 저장 가능

5. 서버 API 보강
   - `/api/mentoring`에 `validateAssignments`, `assignStudents` 액션 추가
   - 반복 요일 대상 차시 자동 매칭
   - 필요한 경우 같은 차시를 다른 요일에 자동 생성

## Supabase

필수 SQL:

```sql
beyond-os-supabase-mentoring-schedule-v41-31.sql
```

기존 v41-30 SQL을 이미 실행했더라도 v41-31 SQL을 한 번 더 실행해야 1~8차시 기본 세팅 및 반복/유효기간 컬럼이 보강됩니다.

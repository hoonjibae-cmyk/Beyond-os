# Beyond OS v41-23 — Public Report Mobile Polish

기준선: v41-22 `beyond-os-v41-22-report-share-link-stability.zip`

## 핵심 변경

1. 학부모 공개 리포트 화면 모바일 가독성 개선
   - 공개 링크(`/r/[token]`) 화면을 학부모 모바일 열람 기준으로 재정리했습니다.
   - The Place 26 로고/브랜드 영역을 상단에 표시합니다.

2. 데일리 공개 리포트 상단 요약 카드 추가
   - 입실
   - 퇴실
   - 총 체류
   - 순공시간
   - 학습상태 기록

3. 문구 통일
   - `외출 제외 학습시간`이 아니라 `순공시간`으로 표기합니다.

4. 차시별 학습 내용 카드형 표시
   - 기존 텍스트형 `오늘의 학습 기록`을 학부모가 읽기 쉬운 카드형 `차시별 학습 내용`으로 표시합니다.
   - 순찰 체크가 속한 차시 전체 구간 기준으로 정리합니다.
   - 파싱이 어려운 기존 리포트는 안전하게 원문형으로 fallback됩니다.

5. 링크 오류/만료 안내 화면 개선
   - 링크가 만료되었거나 비활성화된 경우 문의번호와 함께 안내 카드가 표시됩니다.

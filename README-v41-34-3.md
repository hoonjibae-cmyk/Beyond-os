# Beyond OS v41-34.3

멘토링 시간표의 개인일정 주의 학생 카드 레이아웃 추가 보정 핫픽스입니다.

## 변경 사항

- 개인일정 충돌 학생 카드에서 긴 경고/상세 영역이 학생명 영역을 밀어내지 않도록 2줄 레이아웃으로 재정렬했습니다.
- 경고 표시는 `⚠ 주의 상세` 버튼 하나로 단순화했습니다.
- 학생 이름/멘토/학교 정보, 이동 버튼, `이 날짜에서 제외` 버튼이 같은 줄에서 깨지지 않도록 grid area를 분리했습니다.
- 차시 카드 헤더에서 날짜/요일/시간이 좁은 폭에서 세로로 찢어져 보이지 않도록 재정렬했습니다.
- `요일 기본값 기반`, `배정 변경 있음`, `개인일정 주의 있음` 배지는 차시 제목을 압박하지 않고 헤더 아래쪽에 정렬되도록 조정했습니다.

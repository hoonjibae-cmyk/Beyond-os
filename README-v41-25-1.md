# Beyond OS v41-25.1 — Weekly Report Period Selector Cleanup

## 기준선
- `beyond-os-v41-25-weekly-report-history-search.zip`

## 핵심 수정
1. 위클리 리포트 상단 주차 선택 영역에 `기간 검색` 버튼을 통합했습니다.
   - `이번 주 / 지난 주 / 기간 검색 / 조회` 구조로 정리했습니다.
   - `기간 검색`을 누르면 날짜 선택과 주간 단위 선택지가 포함된 모달이 열립니다.
   - 날짜를 선택하면 해당 날짜가 속한 월~일 주간으로 자동 묶입니다.
   - 최근/과거 주차는 월별로 그룹핑되어 선택할 수 있습니다.

2. 하단 `과거 위클리 리포트 기간 조회`의 중복 빠른 버튼을 제거했습니다.
   - 하단의 `이번 주 / 지난 주 / 최근 4주 / 최근 8주 / 기간 검색` 버튼 묶음을 삭제했습니다.
   - 과거 리포트 영역은 상단에서 선택한 주간의 저장·발송 이력을 보여주는 보조 화면으로 정리했습니다.

3. 위클리 리포트 탭 의미를 정리했습니다.
   - `리포트 작성`: 작성/자동 구성 중심
   - `발송 이력`: 발송 이력 중심
   - `과거 리포트 검색`: 선택 주간의 저장·발송 이력 조회 중심

4. `테스트 번호 대체` 상태 배지를 다듬었습니다.
   - 문구를 `테스트 수신번호 적용`으로 정리했습니다.
   - 초록색 버튼처럼 보이던 상태 표시를 톤앤매너에 맞는 상태 배지로 변경했습니다.

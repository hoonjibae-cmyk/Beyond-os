# Beyond OS v41-32.1 — 출결·관리 이력 완전 통합 화면

## 기준선

- 기준 버전: v41-32
- 패치 성격: 학생 관리 메뉴 내부 탭 제거 및 화면 통합 UX 보정

## 반영 내용

1. `출결 현황 · 멘토 코멘트`와 `학생 관리 이력`을 내부 탭으로 나누지 않고 하나의 화면에 연속 배치했습니다.
2. 상단의 학생/기간 선택 기준으로 아래 누적 관리 이력이 자동 연동됩니다.
3. 멘토링 시간표에서 학생 카드를 클릭하면 기존처럼 통합 메뉴로 이동하고, 오늘 학습멘토 코멘트 입력란으로 포커스됩니다. 같은 학생/기간의 누적 관리 이력도 함께 표시됩니다.
4. 기존 좌측 메뉴명 `출결·관리 이력`은 유지하되, 화면 안의 `출결 현황 · 멘토 코멘트 / 학생 관리 이력` 전환 버튼은 제거했습니다.

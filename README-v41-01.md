# Beyond OS v41-01 - Kiosk Bridge Safety Mode

## 기준
- 기준 파일: beyond-os-v40-123-kiosk-bridge-stability-alerts.zip
- 신규 버전: v41-01
- v40 계열 이후 첫 v41 업데이트
- 신규 SQL: 없음
- 단, v40-115 키오스크 Bridge SQL 필요

## 주요 변경
1. 설정 > 키오스크 브릿지에 자동반영 ON/OFF 안전모드 추가
2. 자동반영 OFF 상태에서는 키오스크 문자 원문과 파싱 결과만 로그에 저장하고 출결/좌석은 변경하지 않음
3. 키오스크 수신 기록이 60분 이상 없으면 확인 필요 경고 표시
4. 오늘 운영 요약과 브릿지 상태 표시 강화
5. /api/kiosk-attendance-bridge가 자동반영 OFF 설정을 인식하도록 보강

## 테스트
1. v41-01 배포 후 상단 버전 확인
2. 설정 > 키오스크 브릿지에서 자동반영 ON 상태 확인
3. 정상 문자 1건 수신 후 자동 출결 반영 확인
4. 자동반영 OFF로 전환 후 문자 1건 수신
5. attendance_import_events에는 로그가 저장되지만 좌석/출결은 변경되지 않는지 확인
6. 다시 ON으로 전환 후 자동반영 정상 재개 확인
7. 마지막 수신 시간이 60분 이상 경과한 상황에서 확인 필요 경고 표시 확인

# Beyond OS v41-22 — Report Share Link Stability

기준선: v41-21 알림톡 실전 발송 전 안전 점검 / 테스트 발송 콘솔

## 핵심 반영 내용

- 데일리 리포트 마감 발송 콘솔에 리포트 링크 생성 상태 요약을 추가했습니다.
- 학생별 카드에서 공개 리포트 링크 상태를 표시합니다.
  - 링크 정상
  - 링크 생성 필요
  - 링크 만료/비활성
  - 리포트 생성 필요
- 발송 가능 학생 또는 전체 발송 대상의 공개 리포트 링크를 미리 생성/재생성할 수 있는 버튼을 추가했습니다.
- 개별 학생 카드에 링크 미리보기와 링크 복사 기능을 정리했습니다.
- 전체 발송 실행 시 발송 직전 학생별 공개 리포트 링크를 자동 생성/재생성한 뒤 발송하도록 보강했습니다.
- 전체 발송 전 확인 모달에 대상별 링크 준비 상태를 표시합니다.
- 데일리 리포트 발송 action log payload에 실제 발송에 사용된 공개 리포트 링크 URL/token을 저장합니다.
- `/api/report-share-link` 조회 API에 `reportType`, `reportIds` 필터와 `linksByReportId` 응답을 추가했습니다.

## 추가 설정

- 신규 SQL 없음
- 신규 환경변수 없음
- MacroDroid 변경 없음

단, 공개 리포트 링크 기능은 기존 v40-83 SQL의 `report_share_links` 테이블이 필요합니다.

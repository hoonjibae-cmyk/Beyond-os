# Beyond OS v41-27 — 관리주의 메뉴 이력화

## 기준선
- v41-26.3 `beyond-os-v41-26-3-seat-card-badge-clipping-fix.zip`

## 반영 내용

### 1. 왼쪽 메뉴명 변경
- 기존 `관리주의 학생` 메뉴를 `관리주의 이력`으로 변경했습니다.
- 실시간 관리주의 처리는 메인 대시보드의 빨간 좌석 / 관리 포커스에서 처리하도록 역할을 분리했습니다.

### 2. 관리주의 이력 조회 화면 추가
- 날짜 범위 조회
- 학생별 필터
- 현재 미처리 / 처리완료 필터
- 학생명, 사유, 메모, 처리자 검색
- 처리완료 이력의 확인 메모 / 처리자 / 처리시각 확인

### 3. 현재 미처리 항목 연결
- 오늘 발생 중인 미처리 관리주의 항목은 이력 화면에서도 참고용으로 표시합니다.
- 실제 처리는 `대시보드에서 처리` 버튼을 통해 메인 대시보드로 이동하도록 정리했습니다.

### 4. 멀티디바이스 이력 기준 유지
- 처리완료 이력은 v41-19에서 추가된 `field_focus_acknowledgements` 테이블을 기준으로 조회합니다.
- 데스크탑/모바일에서 동일한 처리 이력을 확인할 수 있습니다.

# Beyond OS v41-25 — 위클리 리포트 과거 기간 조회

기준선: v41-24 `beyond-os-v41-24-weekly-report-auto-compose.zip`

## 핵심 반영

1. 위클리 리포트 화면에 `과거 리포트 기간 조회` 콘솔 추가
   - 기본 조회 기간은 이번 주로 유지
   - 조회 시작일~종료일을 직접 선택해 과거 리포트 검색 가능
   - 이번 주 / 지난 주 / 최근 4주 / 최근 8주 빠른 조회 버튼 추가

2. 과거 위클리 리포트 검색 결과 표시
   - 학생명, 학교/학년
   - 리포트 기간
   - 저장/수정 시각
   - 총 순공시간, 등원일수
   - 발송 상태
   - 리포트 본문 미리보기

3. 검색/필터 추가
   - 학생 필터
   - 발송상태 필터: 전체, 저장된 리포트, 발송완료, 발송대기, 발송실패, 본문 미작성
   - 학생명/학교/리포트 본문 검색어 필터

4. 과거 리포트 바로 열기
   - 검색 결과에서 `이 기간 리포트 열기` 클릭 시 해당 학생/해당 기간의 위클리 리포트 작성 화면으로 이동
   - 과거 저장 내용과 발송 상태를 그대로 확인 가능

5. API 보강
   - `/api/weekly-report?mode=history&start=YYYY-MM-DD&end=YYYY-MM-DD`
   - 조회 기간과 겹치는 모든 위클리 리포트를 반환
   - 학생 스냅샷(name/school/grade/status/parent_phone)을 함께 반환해 과거 비활성 학생도 식별 가능

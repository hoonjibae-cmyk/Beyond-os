# Beyond OS v41-31.3 — 화·목 임시 멘토링 차시 허용

기준선: v41-31.2

## 반영 내용

1. 화·목 멘토링 차시 추가 허용
   - 기존 기본 세팅은 그대로 월·수·금 1~8차시만 생성합니다.
   - 멘토링 시간표의 요일 탭을 월~금 전체로 확장했습니다.
   - 화요일·목요일 탭에는 `임시` 라벨을 표시하고, 선택 시 `임시 차시 추가` 버튼으로 차시를 만들 수 있습니다.
   - 차시 추가/수정 화면에서도 화요일·목요일을 선택할 수 있습니다.

2. 서버 검증 확장
   - 기존 API에서 월/수/금만 허용하던 요일 검증을 평일(월~금) 허용으로 변경했습니다.
   - 직접 선택 반복 배정에서도 화·목을 포함할 수 있습니다.
   - `월·수·금` 반복 버튼은 기존처럼 월/수/금만 선택합니다.

3. 좌석배치도 표시 연동 유지
   - 화·목 차시에 학생을 배정하면 해당 요일에도 기존과 동일하게 차시 시작 10분 전부터 차시 종료까지 좌석배치도에 파란색 `멘토링` 배지가 표시됩니다.
   - 관리주의 빨간색 우선 정책은 기존과 동일합니다.

## 추가 설정

- 신규 환경변수 없음
- MacroDroid 변경 없음

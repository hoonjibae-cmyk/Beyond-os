# Beyond OS v41-16.1 — 기본 시간표 기반 관리주의 보정 핫픽스

기준선: v41-16

## 수정 배경

v41-16에서는 `student_daily_schedules`에 명시적으로 저장된 개인시간표가 있는 학생만 시간표 불일치 관리주의 판정에 들어갔습니다. 그래서 기본 시간표만 적용되는 학생이 키오스크로 외출/퇴실 처리된 경우, 좌석 상태는 외출/퇴실로 바뀌지만 빨간색 관리주의가 뜨지 않을 수 있었습니다.

## 반영 내용

- 명시적 개인시간표가 없는 학생도 설정 탭의 기본 시간표를 자동 적용해 관리주의 판정에 포함
- 오늘 세션이 있거나 좌석 배정이 있는 학생은 기본 시간표 기준 점검 대상에 포함
- 기본 시간표상 참여해야 하는 시간인데 상태가 외출/퇴실/미입실이면 좌석을 빨간색 관리주의로 우선 표시
- 명시적 개인시간표/외출 예외가 있는 학생은 기존처럼 해당 예외 시간을 우선 적용
- 기존 확인/해제 메모 기능 유지

## 추가 설정

- 신규 SQL 없음
- 신규 환경변수 없음
- MacroDroid 변경 없음

npx.cmd vercel --prod --yes

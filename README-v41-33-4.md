# Beyond OS v41-33.4 — 멘토링 날짜별 일정 안정화

## 기준 버전
- 기준: v41-33.3
- 목적: 날짜별 멘토링 일정 기능을 실제 운영 중 헷갈리지 않게 다듬고, 날짜별 제외/되돌리기/개인일정 주의 확인 흐름을 안정화했습니다.

## 반영 내용

### 1. 날짜별 일정 상태 배지 정리
날짜별 일정 차시 카드에 운영 상태가 더 명확히 표시됩니다.

- 요일 기본값
- 요일 기본값 기반
- 날짜별 수정됨
- 이 날짜에서만 추가됨
- 배정 변경 있음
- 개인일정 주의 있음

### 2. 개인일정 주의 상세 보기
학생 카드에 개인일정 주의가 표시되는 경우, 주의 표시 또는 `주의 상세` 버튼을 눌러 충돌 사유를 확인할 수 있습니다.

상세 팝업에는 다음 정보가 표시됩니다.

- 기준일
- 멘토링 시간
- 학생 개인 일정
- 시간표 메모
- 충돌 사유

### 3. 날짜별 제외 문구 정리
날짜별 일정에서의 삭제/비활성화는 요일 템플릿 삭제가 아니라 해당 날짜에서만 제외하는 의미이므로 문구를 정리했습니다.

- 학생 배정 삭제 → `이 날짜에서 제외`
- 차시 비활성화 → `이 날짜 차시 제외`

요일별 템플릿 화면에서는 기존 의미에 맞게 삭제/비활성화 문구를 유지합니다.

### 4. 중요 작업 확인창 보강
다음 작업은 실행 전 한 번 더 확인합니다.

- 일정 수정 시작
- 기본값 되돌리기
- 이 날짜에서 학생 제외
- 이 날짜 차시 제외

특히 기본값 되돌리기는 이 날짜에서만 추가·수정·제외한 차시와 배정 변경이 초기화된다는 점을 확인 문구에 명시했습니다.

### 5. 작업 피드백 강화
저장, 제외, 되돌리기 작업 중/완료/실패 메시지를 더 명확하게 표시합니다.

- 처리 중 메시지
- 완료 메시지
- 실패 시 새로고침 후 재시도 안내

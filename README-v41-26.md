# Beyond OS v41-26 — 위클리 리포트 UI 정리

기준선: v41-25.1 Weekly Report Period Selector Cleanup

## 반영 내용

1. 위클리 리포트 상단 `기간 검색` 버튼 가독성 개선
   - 버튼이 비활성처럼 흐리게 보이던 문제 수정
   - 색상 대비와 텍스트 가독성 강화
   - `이번 주 / 지난 주 / 기간 검색 / 조회` 버튼 묶음 정렬 보정

2. 주간 선택 버튼 그룹 반응형 정리
   - 데스크탑에서는 4개 버튼을 한 줄로 정렬
   - 좁은 화면에서는 2열로 자연스럽게 줄바꿈
   - `조회` 버튼이 단독으로 어색하게 밀려 내려가는 현상 완화

3. `테스트 수신번호 적용` 상태 배지 정리
   - 기존처럼 큰 버튼처럼 보이지 않도록 축소
   - 위클리 리포트 발송 상태 카드 안에서 작은 상태 라벨처럼 표시
   - 테스트모드/실전모드 상태를 더 자연스럽게 확인 가능

# Beyond OS v41-13 — Field Focus Refinement

기준선: `beyond-os-v41-12-study-quick-check-patrol-mode.zip`

## 핵심 변경

1. 모바일 순찰 퀵체크 입력창 글씨색 수정
   - iOS Safari에서 입력값이 흰색으로 표시되어 보이지 않던 문제 수정
   - `input / textarea / select`의 텍스트 색상, 배경색, placeholder, `-webkit-text-fill-color`를 명시적으로 지정

2. 메인 대시보드 배치 순서 조정
   - 좌석배치도를 `실시간 학습 분포`와 `관리 포커스`보다 위에 배치
   - 현장 관리자가 먼저 좌석 상태를 확인하고, 아래에서 분포/포커스를 보도록 흐름 정리

3. 관리포커스 정교화
   - 결석 학생은 관리포커스 명단에서 제외
   - 단순 외출/단순 미입실도 관리포커스에서 제외
   - 관리포커스는 현장에서 즉시 확인할 필요가 있는 항목 중심으로 정리

4. 결석 색상 변경
   - 결석 상태는 빨간색이 아니라 보라색 계열로 표시
   - 빨간색은 현장 확인이 필요한 긴급 상태 전용으로 사용

5. 긴급 확인 필요 좌석 빨간색 우선 표시
   - 출결 상태 색상보다 우선해서 빨간색 적용
   - 대상 예시:
     - 외출 복귀 예정시간 경과
     - 개인 시간표상 있어야 하는데 입실 기록이 없는 경우
   - 빨간색은 문제 해결 또는 관리자 확인 처리 후 사라지도록 구성

6. 집중관리대상 해제 기능
   - 관리포커스의 긴급 항목에 `확인/해제` 버튼 추가
   - 관리자가 확인 메모를 입력하면 해당 항목은 당일 집중관리대상에서 해제
   - 해제 내역은 브라우저 localStorage에 당일 기준으로 저장

# Beyond OS v41-16 — 시간표 참여시간 관리주의 보정

기준선: `beyond-os-v41-15-daily-report-closeout-send-console.zip`

## 핵심 수정

- 개인시간표상 현재 참여해야 하는 시간인데 출결 상태가 `외출`, `퇴실`, `미입실`인 학생을 관리주의 빨간색으로 우선 표시합니다.
- 기존에는 `미입실` 중심으로만 시간표 불일치가 표시되어, 키오스크에서 `외출` 또는 `퇴실`로 변경된 학생이 빨간색 관리주의로 뜨지 않는 문제가 있었습니다.
- 등록된 외출 일정의 외출 시간 안에 있는 학생은 정상 예외로 보고 빨간색 표시하지 않습니다.
- 등록된 복귀 예정시간을 넘겼거나, 시간표 참여시간 중 외출/퇴실 상태이면 빨간색 관리주의 표시 및 관리포커스에 노출합니다.
- 관리자가 확인/해제 메모를 입력하면 기존 v41-13 방식대로 당일 집중관리대상에서 해제됩니다.

- 신규 Vercel 환경변수 없음
- MacroDroid 변경 없음

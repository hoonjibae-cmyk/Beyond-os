# Beyond OS v41-32.2 — 학생 관리 흐름 연결성 보정

## 기준선

- 기준 버전: v41-32.1
- 패치 성격: 멘토링 시간표 → 출결·관리 이력 → 코멘트 저장 흐름 UX 보정

## 반영 내용

1. 코멘트 저장 후 멘토링 시간표로 돌아가기
   - 멘토링 시간표에서 학생 카드를 클릭해 `출결·관리 이력`으로 이동한 경우, 오늘 학습멘토 코멘트를 저장하면 `멘토링 시간표로 돌아가기` 버튼이 표시됩니다.
   - 저장 직후 다음 학생을 확인하기 위해 시간표로 되돌아가는 동선을 줄였습니다.

2. 같은 차시 이전/다음 학생 이동
   - 멘토링 시간표에서 같은 차시에 배정된 학생 목록을 함께 전달합니다.
   - `출결·관리 이력`의 오늘 학습멘토 코멘트 영역에서 같은 차시의 이전 학생 / 다음 학생으로 바로 이동할 수 있습니다.
   - 이동 시 해당 학생이 자동 선택되고, 조회 기간은 오늘로 맞춰지며, 오늘 코멘트 입력란에 다시 포커스됩니다.

3. 오늘 기록 / 누적 기록 UI 구분 강화
   - 통합 화면 상단에 `오늘 기록` 영역을 두고, 출결 현황과 오늘 학습멘토 코멘트를 먼저 확인하도록 정리했습니다.
   - 아래에는 `누적 기록` 영역을 별도로 구분해 학생 관리 이력이 이어서 표시되도록 했습니다.
   - 내부 탭은 추가하지 않고 한 화면 구조는 그대로 유지했습니다.

4. 멘토링 시간표 복귀 시 요일 유지
   - 멘토링 시간표로 돌아갈 때 학생을 클릭했던 차시의 요일 탭이 우선 열리도록 보정했습니다.

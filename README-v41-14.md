# Beyond OS v41-14 — 대시보드 정리 및 모바일 사용성 개선

기준선: `beyond-os-v41-13-field-focus-refinement.zip`

## 핵심 변경

1. **상단 중복 써머리카드 삭제**
   - 대시보드 상단의 `전체 좌석 / 현재 재실 / 외출 / 미입실 / 퇴실 / 결석` 요약 카드 묶음을 삭제했습니다.
   - 좌석배치도 내부의 `전체 / 입실 / 외출 / 미입실 / 퇴실 / 결석 / 관리필요` 필터 카드만 유지합니다.

2. **좌석배치도 중심 화면 구조 정리**
   - 화면 순서를 `좌석배치도 → 순찰 퀵체크 → 실시간 학습 분포 → 관리 포커스`로 정리했습니다.
   - 현장 관리자가 먼저 전체 좌석 상태를 보고, 필요 시 순찰 모드를 켜는 흐름으로 맞췄습니다.

3. **모바일 좌석배치도 사용성 개선**
   - 모바일에서 좌석 필터 버튼을 가로 스크롤 형태로 정리했습니다.
   - 검색창과 좌석맵의 터치/스크롤 영역을 개선했습니다.
   - 좌석맵은 기존 좌표 배치를 유지하면서 모바일에서 가로 스크롤로 안정적으로 볼 수 있게 보정했습니다.

4. **관리필요 해제 이력 표시**
   - 관리 포커스 카드 하단에 `오늘 확인/해제 이력`을 표시합니다.
   - 해제 메모, 담당자, 확인 시간을 간단히 볼 수 있습니다.

## 반영 내용
1. 현재 출결 상태와 맞지 않는 키오스크 이벤트 자동 차단
   - 입실 기록 없는 외출/복귀/퇴실 차단
   - 외출 상태가 아닌 복귀 차단
   - 이미 퇴실한 상태에서 중복 하원 문자 무시
   - 이미 입실/외출 상태인 중복 문자를 중복 무시로 기록
2. 퇴실 후 다시 도착 문자는 재입실 흐름으로 허용
3. 관리자 수동 처리 후 60초 이내 동일 키오스크 이벤트가 들어오면 중복 무시
4. 키오스크 로그 상태 문구를 운영자용으로 정리
   - 자동반영 완료 / 자동반영 실패 / 중복 무시 / 자동반영 보류
5. 키오스크 브릿지 상태 카드에 성공률과 운영 상태 문구 보강
6. 설정 > 키오스크 브릿지에 문자 원문 파서 테스트 도구 추가
   - 실제 출결 반영 없이 학생명/출결유형/사유 추출 결과 확인 가능

## 테스트
1. 메인 상단에 v41-03 표시 확인
2. 설정 > 키오스크 브릿지 > 문자 원문 파서 테스트에서 샘플 문자 분석
3. 입실 기록 없는 학생에게 외출 문자 테스트 → 자동반영 실패로 기록되는지 확인
4. 외출 상태가 아닌 학생에게 복귀 문자 테스트 → 자동반영 실패로 기록되는지 확인
5. 좌석배치도에서 수동 입실 처리 후 60초 이내 동일 입실 문자 전송 → 중복 무시로 기록되는지 확인
6. 정상 입실/외출/복귀/퇴실 문자는 기존처럼 자동반영되는지 확인

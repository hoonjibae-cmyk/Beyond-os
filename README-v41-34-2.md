# Beyond OS v41-34.2

멘토링 개인일정 주의 카드 레이아웃 핫픽스입니다.

## 변경 사항

1. 학생 카드 내부의 긴 경고 문구 제거
   - 기존 `⚠ 개인일정 주의 · 사유 보기` 문구가 좁은 카드에서 세로로 깨지는 문제를 방지했습니다.

2. 경고 표시 단순화
   - 개인일정 충돌이 있는 학생은 `⚠` 아이콘과 `주의 상세` 버튼만 표시됩니다.
   - `주의 상세` 버튼을 눌러 기존 상세 팝업을 확인할 수 있습니다.

3. 레이아웃 보정
   - 이동 핸들, 학생 정보, 경고 아이콘, 주의 상세 버튼, 이 날짜에서 제외 버튼이 한 줄에서 더 안정적으로 정렬되도록 CSS를 보정했습니다.

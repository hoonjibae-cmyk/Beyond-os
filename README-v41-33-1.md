# v41-33.1 — 멘토링 날짜별 일정 수정 UX 핫픽스

## 반영 사항

1. 날짜별 일정 수정 모드 삭제 동작 보강
   - `선택 날짜 일정 수정 시작` 이후 이미 배정된 학생의 `삭제` 버튼이 안정적으로 동작하도록 보강했습니다.
   - 화면에 남아 있는 가상 템플릿 ID/요일 템플릿 ID도 서버에서 날짜별 실제 일정으로 변환 후 처리합니다.

2. 날짜별 차시 추가 UI 개선
   - 날짜별 차시 추가 시 차시명/시작시간/종료시간을 수기로 입력하는 대신 `1~8차시` 드롭다운에서 선택하도록 정리했습니다.
   - 선택한 차시의 시작/종료 시간은 자동 입력됩니다.

3. 권장 인원 입력 설명 추가
   - 숫자 `3`, `4`만 보이던 입력칸에 각각 `권장 최소 인원`, `권장 최대 인원` 라벨을 추가했습니다.
   - 차시별 적정 배정 인원 표시 기준이라는 설명을 함께 표시합니다.
